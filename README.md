# Skeleton

My skeleton for building single page applications.

## Ubuntu/Windows start script

- Ubuntu:

```json
"start": "webpack serve --open /usr/bin/google-chrome"
```

- Windows:

```json
"start": "webpack serve --open chrome"
```
